#### 核心功能
- ✅ **Recording Agent**

  - 时间推理（今天/昨天/前天/早上/中午/晚上）
  - 金额提取（数字+单位/纯数字）
  - 智能分类（9大类别+子类别）
  - 多笔交易解析

- ✅ **REST API**
  - POST /api/v1/transactions/parse - 解析自然语言

  - GET /api/v1/transactions - 查询交易列表
  - GET /api/v1/transactions/{id} - 查询单个交易
  - DELETE /api/v1/transactions/{id} - 删除交易
  - GET /health - 健康检查

## 核心创新点 🌟

### 1. 混合推理引擎
- **规则引擎优先**: 0延迟，0成本
- **LLM Fallback**: confidence < 0.8时调用

- 细分子类别
- 关键词匹配 + LLM fallback

## API示例 💡

### 解析交易
```bash
curl -X POST "http://localhost:8000/api/v1/transactions/parse" \
  -H "Content-Type: application/json" \
  -d '{"input": "今天午饭花了35块"}'

# 响应
{
  "success": true,
  "transactions": [{
    "amount": 35.0,
    "category": "餐饮美食",
    "subcategory": "午餐",
    "transaction_date": "2026-03-30",
    "confidence": 0.95
  }],
  "message": "已识别：餐饮美食 - 午餐，35.0元"
}
```

## 运行指南 🚀

### 后端启动
```bash
cd backend

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑.env，添加OPENAI_API_KEY

# 启动服务
python main.py

# 访问API文档
open http://localhost:8000/docs
```

### 移动端启动
```bash
cd mobile

# 安装依赖
npm install

# 启动Expo
npx expo start

# 在模拟器运行
# iOS: 按 i
# Android: 按 a
```
