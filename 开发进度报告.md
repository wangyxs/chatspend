## 📝 技术决策

### 1. 后端驱动渲染架构
**决策**: 后端决定UI组件，前端只渲染  
**参考**: 王自如AI产品设计理念

**优势**:
- 前端逻辑简化
- 后端灵活控制展示
- 便于A/B测试
- 支持多端统一逻辑

**实现**:
```python
# 后端构建UI组件
builder = ResponseBuilder()
response = builder.build_transaction_created_response(transaction)
return response.model_dump()
```

```typescript
// 前端渲染组件
{components.map((component) => (
  <UIComponentRenderer component={component} onAction={handleAction} />
))}
```

### 2. UI组件化设计
**决策**: 10种标准化UI组件  
**能力**:
- 交易展示：TRANSACTION_CARD
- 数据概览：SUMMARY_CARD + CHART_CARD
- 智能洞察：INSIGHT_CARD
- 预算管理：BUDGET_CARD
- 交互引导：QUICK_ACTIONS

### 3. 无缝对话体验
**决策**: 像微信一样的对话流  
**特点**:
- 持续输入需求
- 多意图批量操作
- 跨业务域切换
- 快捷操作建议
