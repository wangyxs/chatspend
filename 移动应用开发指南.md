# ChatSpend 移动应用开发指南

本文档详细介绍如何使用 React Native + Expo 开发 ChatSpend 移动应用，并最终上架到 App Store 和各大 Android 应用商店。

---

## 一、技术栈选择理由

### 1.1 为什么选择 React Native + Expo？

**React Native 优势**:
- ✅ 一套代码同时支持 iOS 和 Android
- ✅ 接近原生的性能体验
- ✅ 丰富的第三方库生态
- ✅ 热更新支持（CodePush）
- ✅ JavaScript/TypeScript 技术栈，学习成本低

**Expo 优势**:
- ✅ 开箱即用，无需配置原生开发环境
- ✅ 提供丰富的 API（相机、语音、推送等）
- ✅ OTA 更新支持
- ✅ EAS Build 云端构建，无需本地 Xcode/Android Studio
- ✅ 快速原型开发

**适用场景**:
- ✅ 中小型应用（ChatSpend 属于此类）
- ✅ 需要快速迭代
- ✅ 团队熟悉 JavaScript/TypeScript
- ✅ 不需要深度原生定制

```typescript
  Platform,
  StyleSheet,

import { MessageBubble } from '../components/MessageBubble';
import { useTransactionStore } from '../stores/transactionStore';
import { useVoiceInput } from '../hooks/useVoiceInput';
import { useCamera } from '../hooks/useCamera';
import { api } from '../services/api';

interface Message {
  id: string;
  type: 'user' | 'assistant';
  content: string;
  timestamp: Date;
  transaction?: Transaction;
}

export const HomeScreen: React.FC = () => {
  const [messages, setMessages] = useState<Message[]>([]);
  const [inputText, setInputText] = useState('');
  const [isLoading, setIsLoading] = useState(false);
  const flatListRef = useRef<FlatList>(null);
  
  const addTransaction = useTransactionStore(state => state.addTransaction);
  const { startRecording, stopRecording, isRecording } = useVoiceInput({
    onResult: (text) => handleSendMessage(text),
  });
  const { takePhoto, pickImage } = useCamera({
    onCapture: handleImageCapture,
  });
  
  // 发送文本消息
  const handleSendMessage = async (text: string) => {
    if (!text.trim()) return;
    
    // 添加用户消息
    const userMessage: Message = {
      id: Date.now().toString(),
      type: 'user',
      content: text,
      timestamp: new Date(),
    };
    setMessages(prev => [...prev, userMessage]);
    setInputText('');
    
    setIsLoading(true);
    
    try {
      // 调用API解析交易
      const result = await api.parseTransaction(text);
      
      // 保存交易
      if (result.transactions) {
        for (const transaction of result.transactions) {
          await addTransaction(transaction);
        }
      }
      
      // 添加助手回复
      const assistantMessage: Message = {
        id: (Date.now() + 1).toString(),
        type: 'assistant',
        content: result.message,
        timestamp: new Date(),
        transaction: result.transactions?.[0],
      };
      setMessages(prev => [...prev, assistantMessage]);
      
      // 滚动到底部
      setTimeout(() => {
        flatListRef.current?.scrollToEnd();
      }, 100);
      
    } catch (error) {
      console.error('Failed to send message:', error);
      // 错误提示
      const errorMessage: Message = {
        id: (Date.now() + 1).toString(),
        type: 'assistant',
        content: '抱歉，处理失败了。请重试。',
        timestamp: new Date(),
      };
      setMessages(prev => [...prev, errorMessage]);
    } finally {
      setIsLoading(false);
    }
  };
  
  // 处理图片识别
  const handleImageCapture = async (photo: string) => {
    setIsLoading(true);
    
    try {
      const result = await api.recognizeReceipt(photo);
      
      // 添加用户消息（图片）
      const userMessage: Message = {
        id: Date.now().toString(),
        type: 'user',
        content: '📷 上传了小票',
        timestamp: new Date(),
      };
      setMessages(prev => [...prev, userMessage]);
      
      // 添加助手回复
      const assistantMessage: Message = {
        id: (Date.now() + 1).toString(),
        type: 'assistant',
        content: result.message,
        timestamp: new Date(),
        transaction: result.transaction,
      };
      setMessages(prev => [...prev, assistantMessage]);
      
    } catch (error) {
      console.error('Failed to recognize receipt:', error);
    } finally {
      setIsLoading(false);
    }
  };
  
  return (
    <KeyboardAvoidingView
      style={styles.container}
      behavior={Platform.OS === 'ios' ? 'padding' : 'height'}
    >
      {/* 消息列表 */}
      <FlatList
        ref={flatListRef}
        data={messages}
        keyExtractor={item => item.id}
        renderItem={({ item }) => <MessageBubble message={item} />}
        contentContainerStyle={styles.messageList}
        onContentSizeChange={() => flatListRef.current?.scrollToEnd()}
      />
      
      {/* 输入框 */}
      <View style={styles.inputContainer}>
        {/* 相机按钮 */}
        <TouchableOpacity
          style={styles.iconButton}
          onPress={takePhoto}
        >
          <Ionicons name="camera" size={24} color="#666" />
        </TouchableOpacity>
        
        {/* 相册按钮 */}
        <TouchableOpacity
          style={styles.iconButton}
          onPress={pickImage}
        >
          <Ionicons name="image" size={24} color="#666" />
        </TouchableOpacity>
        
        {/* 文本输入 */}

          style={styles.input}
          value={inputText}
          onChangeText={setInputText}
          placeholder="说点什么或输入金额..."
          multiline

        />

    </KeyboardAvoidingView>
  );
};
```

### 3.2 语音输入 Hook

```typescript
// src/hooks/useVoiceInput.ts
import { useState, useCallback } from 'react';
import * as FileSystem from 'expo-file-system';
import { Audio } from 'expo-av';

interface UseVoiceInputProps {
  onResult: (text: string) => void;
  onError?: (error: Error) => void;
}

export const useVoiceInput = ({ onResult, onError }: UseVoiceInputProps) => {
  const [isRecording, setIsRecording] = useState(false);
  const [recording, setRecording] = useState<Audio.Recording | null>(null);
  
  const startRecording = useCallback(async () => {
    try {
      // 请求权限
      const { status } = await Audio.requestPermissionsAsync();
      if (status !== 'granted') {
        throw new Error('Microphone permission not granted');
      }
      
      // 配置音频
      await Audio.setAudioModeAsync({
        allowsRecordingIOS: true,
        playsInSilentModeIOS: true,
      });
      
      // 开始录音
      const { recording } = await Audio.Recording.createAsync(
        Audio.RecordingOptionsPresets.HIGH_QUALITY
      );
      setRecording(recording);
      setIsRecording(true);
      
    } catch (error) {
      console.error('Failed to start recording:', error);
      onError?.(error as Error);
    }
  }, [onError]);
  
  const stopRecording = useCallback(async () => {
    if (!recording) return;
    
    try {
      setIsRecording(false);
      await recording.stopAndUnloadAsync();
      
      // 获取录音文件URI
      const uri = recording.getURI();
      setRecording(null);
      
      if (uri) {
        // 上传到服务器进行语音识别
        const text = await recognizeSpeech(uri);
        onResult(text);
      }
      
    } catch (error) {
      console.error('Failed to stop recording:', error);
      onError?.(error as Error);
    }
  }, [recording, onResult, onError]);
  
  // 调用语音识别API
  const recognizeSpeech = async (uri: string): Promise<string> => {
    // 读取音频文件
    const audioData = await FileSystem.readAsStringAsync(uri, {
      encoding: FileSystem.EncodingType.Base64,
    });
    
    // 调用后端API（OpenAI Whisper）
    const response = await fetch('https://api.chatspend.app/api/v1/voice/recognize', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        audio: audioData,
      }),
    });
    
    const result = await response.json();
    return result.text;
  };
  
  return {
    isRecording,
    startRecording,
    stopRecording,
  };
};
```

### 3.3 相机拍照 Hook

```typescript
// src/hooks/useCamera.ts
import { useState } from 'react';
import * as ImagePicker from 'expo-image-picker';
import * as FileSystem from 'expo-file-system';

interface UseCameraProps {
  onCapture: (photo: string) => void;
  onError?: (error: Error) => void;
}

export const useCamera = ({ onCapture, onError }: UseCameraProps) => {
  const [isLoading, setIsLoading] = useState(false);
  
  // 拍照
  const takePhoto = async () => {
    try {
      // 请求相机权限
      const { status } = await ImagePicker.requestCameraPermissionsAsync();
      if (status !== 'granted') {
        throw new Error('Camera permission not granted');
      }
      
      // 打开相机
      const result = await ImagePicker.launchCameraAsync({
        mediaTypes: ImagePicker.MediaTypeOptions.Images,
        allowsEditing: true,
        aspect: [4, 3],
        quality: 0.8,
        base64: true,
      });
      
      if (!result.canceled && result.assets[0]) {
        setIsLoading(true);
        
        // 获取图片Base64
        const base64 = result.assets[0].base64;
        if (base64) {
          onCapture(`data:image/jpeg;base64,${base64}`);
        }
        
        setIsLoading(false);
      }
      
    } catch (error) {
      console.error('Failed to take photo:', error);
      onError?.(error as Error);
      setIsLoading(false);
    }
  };
  
  // 从相册选择
  const pickImage = async () => {
    try {
      // 请求相册权限
      const { status } = await ImagePicker.requestMediaLibraryPermissionsAsync();
      if (status !== 'granted') {
        throw new Error('Media library permission not granted');
      }
      
      // 打开相册
      const result = await ImagePicker.launchImageLibraryAsync({
        mediaTypes: ImagePicker.MediaTypeOptions.Images,
        allowsEditing: true,
        aspect: [4, 3],
        quality: 0.8,
        base64: true,
      });
      
      if (!result.canceled && result.assets[0]) {
        setIsLoading(true);
        
        // 获取图片Base64
        const base64 = result.assets[0].base64;
        if (base64) {
          onCapture(`data:image/jpeg;base64,${base64}`);
        }
        
        setIsLoading(false);
      }
      
    } catch (error) {
      console.error('Failed to pick image:', error);
      onError?.(error as Error);
      setIsLoading(false);
    }
  };
  
  return {
    takePhoto,
    pickImage,
    isLoading,
  };
};
```

```typescript
// src/services/storage.ts
import * as SQLite from 'expo-sqlite';
import * as SecureStore from 'expo-secure-store';

const db = SQLite.openDatabase('chatspend.db');

// 初始化数据库
export const initDatabase = (): Promise<void> => {
  return new Promise((resolve, reject) => {
    db.transaction(tx => {
      // 交易表
      tx.executeSql(`
        CREATE TABLE IF NOT EXISTS transactions (
          id TEXT PRIMARY KEY,
          amount REAL NOT NULL,
          category TEXT NOT NULL,
          subcategory TEXT,
          date TEXT NOT NULL,
          time TEXT,
          description TEXT,
          merchant TEXT,
          payment_method TEXT,
          tags TEXT,
          confidence_score REAL,
          is_confirmed INTEGER DEFAULT 0,
          created_at TEXT NOT NULL,
          updated_at TEXT NOT NULL,
          synced_at TEXT
        )
      `);
      
      // 预算表
      tx.executeSql(`
        CREATE TABLE IF NOT EXISTS budgets (
          id TEXT PRIMARY KEY,
          category TEXT,
          amount REAL NOT NULL,
          period TEXT NOT NULL,
          start_date TEXT NOT NULL,
          end_date TEXT,
          is_active INTEGER DEFAULT 1,
          created_at TEXT NOT NULL,
          updated_at TEXT NOT NULL
        )
      `);
      
      // 用户设置表
      tx.executeSql(`
        CREATE TABLE IF NOT EXISTS settings (
          key TEXT PRIMARY KEY,
          value TEXT NOT NULL,
          updated_at TEXT NOT NULL
        )
      `);
      
    }, reject, resolve);
  });
};

// 交易相关操作
export const transactionStorage = {
  // 插入交易
  insert: async (transaction: Transaction): Promise<void> => {
    return new Promise((resolve, reject) => {
      db.transaction(tx => {
        tx.executeSql(
          `INSERT INTO transactions (
            id, amount, category, subcategory, date, time, description,
            merchant, payment_method, tags, confidence_score, is_confirmed,
            created_at, updated_at
          ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)`,
          [
            transaction.id,
            transaction.amount,
            transaction.category,
            transaction.subcategory,
            transaction.date,
            transaction.time,
            transaction.description,
            transaction.merchant,
            transaction.payment_method,
            JSON.stringify(transaction.tags),
            transaction.confidence_score,
            transaction.is_confirmed ? 1 : 0,
            new Date().toISOString(),
            new Date().toISOString(),
          ]
        );
      }, reject, resolve);
    });
  },
  
  // 查询交易
  getAll: async (limit = 100, offset = 0): Promise<Transaction[]> => {
    return new Promise((resolve, reject) => {
      db.transaction(tx => {
        tx.executeSql(
          `SELECT * FROM transactions 
           ORDER BY date DESC, created_at DESC 
           LIMIT ? OFFSET ?`,
          [limit, offset],
          (_, { rows }) => {
            const transactions: Transaction[] = [];
            for (let i = 0; i < rows.length; i++) {
              transactions.push(rows.item(i));
            }
            resolve(transactions);
          }
        );
      }, reject);
    });
  },
  
  // 按日期范围查询
  getByDateRange: async (startDate: string, endDate: string): Promise<Transaction[]> => {
    return new Promise((resolve, reject) => {
      db.transaction(tx => {
        tx.executeSql(
          `SELECT * FROM transactions 
           WHERE date >= ? AND date <= ? 
           ORDER BY date DESC`,
          [startDate, endDate],
          (_, { rows }) => {
            const transactions: Transaction[] = [];
            for (let i = 0; i < rows.length; i++) {
              transactions.push(rows.item(i));
            }
            resolve(transactions);
          }
        );
      }, reject);
    });
  },
  
  // 更新交易
  update: async (id: string, updates: Partial<Transaction>): Promise<void> => {
    const fields = Object.keys(updates).map(key => `${key} = ?`).join(', ');
    const values = [...Object.values(updates), new Date().toISOString(), id];
    
    return new Promise((resolve, reject) => {
      db.transaction(tx => {
        tx.executeSql(
          `UPDATE transactions SET ${fields}, updated_at = ? WHERE id = ?`,
          values
        );
      }, reject, resolve);
    });
  },
  
  // 删除交易（软删除）
  delete: async (id: string): Promise<void> => {
    return new Promise((resolve, reject) => {
      db.transaction(tx => {
        tx.executeSql(
          `UPDATE transactions SET deleted_at = ? WHERE id = ?`,
          [new Date().toISOString(), id]
        );
      }, reject, resolve);
    });
  },
  
  // 获取未同步的交易
  getUnsynced: async (): Promise<Transaction[]> => {
    return new Promise((resolve, reject) => {
      db.transaction(tx => {
        tx.executeSql(
          `SELECT * FROM transactions WHERE synced_at IS NULL`,
          [],
          (_, { rows }) => {
            const transactions: Transaction[] = [];
            for (let i = 0; i < rows.length; i++) {
              transactions.push(rows.item(i));
            }
            resolve(transactions);
          }
        );
      }, reject);
    });
  },
};
```

```typescript
  // 请求权限
  requestPermissions: async (): Promise<boolean> => {
    const { status: existingStatus } = await Notifications.getPermissionsAsync();
    let finalStatus = existingStatus;
    
    if (existingStatus !== 'granted') {
      const { status } = await Notifications.requestPermissionsAsync();
      finalStatus = status;
    }
    
    if (finalStatus !== 'granted') {
      return false;
    }
    
    // iOS: 获取推送令牌
    if (Platform.OS === 'ios') {
      const { data: token } = await Notifications.getExpoPushTokenAsync();
      console.log('Push token:', token);
      // 将token发送到服务器
    }
    
    return true;
  },
  
  // 配置通知频道（Android）
  configureChannels: async () => {
    if (Platform.OS === 'android') {
      await Notifications.setNotificationChannelAsync('reminders', {
        name: '提醒',
        importance: Notifications.AndroidImportance.HIGH,
        vibrationPattern: [0, 250, 250, 250],
        lightColor: '#007AFF',
      });
      
      await Notifications.setNotificationChannelAsync('budget', {
        name: '预算提醒',
        importance: Notifications.AndroidImportance.HIGH,
        vibrationPattern: [0, 250, 250, 250],
        lightColor: '#FF9500',
      });
    }
  },
  
  // 调度每日提醒
  scheduleDailyReminder: async (hour: number = 20, minute: number = 0) => {
    await Notifications.cancelAllScheduledNotificationsAsync();
    
    await Notifications.scheduleNotificationAsync({
      content: {
        title: '📊 今日消费提醒',
        body: '快来看看今天的消费情况吧！',
        sound: 'default',
      },
      trigger: {
        hour,
        minute,
        repeats: true,
      },
    });
  },
  
  // 发送预算预警
  sendBudgetAlert: async (category: string, percentage: number) => {
    let message = '';
    if (percentage >= 100) {
      message = `⚠️ ${category}预算已超支！`;
    } else if (percentage >= 80) {
      message = `⚠️ ${category}预算已使用${percentage}%，请注意控制`;
    } else {
      message = `${category}预算已使用${percentage}%`;
    }
    
    await Notifications.scheduleNotificationAsync({
      content: {
        title: '预算提醒',
        body: message,
        sound: 'default',
      },
      trigger: null, // 立即发送
    });
  },
  
  // 发送大额消费提醒
  sendLargeExpenseAlert: async (amount: number, description: string) => {
    await Notifications.scheduleNotificationAsync({
      content: {
        title: '💰 大额消费提醒',
        body: `检测到消费${amount}元：${description}`,
        sound: 'default',
      },
      trigger: null,
    });
  },
  
  // 监听通知
  addListener: (callback: (notification: Notifications.Notification) => void) => {
    return Notifications.addNotificationReceivedListener(callback);
  },
  
  // 监听通知点击
  addResponseListener: (callback: (response: Notifications.NotificationResponse) => void) => {
    return Notifications.addNotificationResponseReceivedListener(callback);
  },
};
```

```typescript
const TransactionList: React.FC<{ transactions: Transaction[] }> = ({ transactions }) => {

    <TransactionCard
      transaction={item}
      onPress={() => handleTransactionPress(item)}
    />
  ), [handleTransactionPress]);
  
  return (
    <FlatList
      data={sortedTransactions}
      renderItem={renderItem}
```

```typescript
export const performanceMonitor = {

    const startTime = Date.now();
    
    try {
      const result = await request();
      const responseTime = Date.now() - startTime;

      }
      
      return result;
    } catch (error) {

      console.error(`API ${endpoint} failed after ${failTime}ms:`, error);
      throw error;
    }
```

## 七、常见问题与解决方案

### 7.1 iOS 特有问题

**问题**: App Store 审核拒绝 - 缺少 Sign in with Apple
**解决**: 如果应用提供第三方登录（微信、Google），必须提供 Sign in with Apple

```typescript
import * as AppleAuthentication from 'expo-apple-authentication';

const handleAppleSignIn = async () => {
  try {
    const credential = await AppleAuthentication.signInAsync({
      requestedScopes: [
        AppleAuthentication.AppleAuthenticationScope.FULL_NAME,
        AppleAuthentication.AppleAuthenticationScope.EMAIL,
      ],
    });
    
    // 使用 credential.identityToken 登录
  } catch (error) {
    console.error('Apple Sign In failed:', error);
  }
};
```

### 7.2 Android 特有问题

**问题**: 国内应用商店需要软件著作权
**解决**: 提前申请软件著作权，或找代理公司办理

**问题**: 不同厂商推送服务不兼容
**解决**: 集成各厂商推送SDK，或使用统一推送服务（个推、极光）

### 7.3 性能问题

**问题**: 长列表滚动卡顿
**解决**: 
- 使用 FlatList 虚拟化
- 优化 renderItem 函数
- 使用 react-native-reanimated

**问题**: 内存泄漏
**解决**: 
- 在 useEffect 清除定时器和监听器
- 避免闭包陷阱
- 使用 useMemo 和 useCallback
