# 2026-03-30 工作日志

## 项目启动：ChatSpend - 自然语言记账Agent应用

### 需求确认
用户选择开发**个人记账助手**应用，核心场景包括：
- 记账方式：自然语言输入、语音记账、图片识别记账、与微信/支付宝账单集成
- 智能分析：消费分类洞察、异常消费提醒、理财建议生成
- 交互体验：对话机器人交互、快捷指令、定期主动汇报、多轮对话修正
- 数据存储：本地存储、云端同步、支持导出报表

**重要更新**: 用户要求上架App Store和Android应用商店，技术栈调整为移动应用开发。

### 架构设计
参考王自如的Agent架构理念，采用：
1. **领域驱动设计（DDD）** - 4个独立业务域：交易管理、分析洞察、预算管理、提醒服务
2. **多智能体架构** - 5个专项Agent：Orchestrator、Recording、Analysis、Budget、Reminder
3. **Plan-and-Execute框架** - 先生成计划，再批量执行，提升效率
4. **Smart Agent, Dumb Tools** - Agent负责推理决策，工具只做执行

### 技术栈选型（移动应用）
- **后端**: Python 3.11+ / FastAPI / LangChain或AutoGen / SQLAlchemy / SQLite+PostgreSQL
- **移动端**: React Native 0.73+ / Expo / TypeScript 5.0+
  - 导航: React Navigation 6.x
  - 状态管理: Zustand
  - 图表: React Native Chart Kit / Victory Native
  - 本地存储: expo-sqlite + AsyncStorage
  - 推送: Expo Notifications / Firebase Cloud Messaging
- **AI能力**: OpenAI GPT-4 / Whisper / GPT-4 Vision
- **部署**: 
  - iOS: App Store Connect + TestFlight
  - Android: Google Play + 国内应用商店（应用宝、华为、小米、OPPO、vivo）

### 核心创新点
1. **时间推理模块** - 智能解析"昨天"、"前天"、"上周一"等模糊时间表达
2. **混合分类推理** - 规则引擎 + LLM，兼顾速度和准确性
3. **多模态处理** - 支持文本、语音、图片三种输入方式
4. **智能提醒系统** - 时间触发、事件触发、条件触发三种机制
5. **个性化建议** - 基于消费数据的理财建议和预算优化
6. **离线优先** - 移动端本地存储，云端同步作为备份
